# Feed 内容评分标准 (10分制)

## 评分维度

### 1. 内容价值 (40%)
- **9-10分**: 原创深度分析、系统性教程、重大新闻、开源项目发布
- **7-8分**: 实用技术方案、独特见解、有价值的经验分享
- **5-6分**: 常见问题讨论、转载但有补充
- **3-4分**: 简单问答、纯信息转发
- **1-2分**: 灌水、广告、无实质内容

### 2. 时效性 (30%)
- **9-10分**: 突发新闻、重大更新、限时信息
- **7-8分**: 近 24 小时热点
- **5-6分**: 近一周话题
- **3-4分**: 常规/持续性话题
- **1-2分**: 过时信息

### 3. 相关性 (20%)
- **9-10分**: 与技术/AI/开发高度相关
- **7-8分**: 科技领域相关
- **5-6分**: 泛知识领域
- **3-4分**: 生活/娱乐
- **1-2分**: 无关紧要

### 4. 可读性 (10%)
- **9-10分**: 标题清晰、描述完整
- **7-8分**: 基本可理解
- **5-6分**: 需要点击查看
- **3-4分**: 标题党
- **1-2分**: 无法理解

## 综合评分

```
总分 = 内容价值*0.4 + 时效性*0.3 + 相关性*0.2 + 可读性*0.1
```

## 筛选规则

- **≥7分**: 精选推荐，详细总结
- **4-6分**: 值得一看，简要描述
- **<4分**: 忽略不显示

## 自动分类参考

根据内容智能分类，常见类别：
- **技术开发**: 编程、框架、工具、开源
- **AI/ML**: 人工智能、大模型、机器学习
- **云计算/运维**: 服务器、DevOps、容器
- **科技资讯**: 行业新闻、产品发布
- **数码硬件**: 设备、配件、评测
- **职场/生活**: 工作、学习、生活
- **其他**: 无法明确分类的内容

分类由 Claude 根据标题和描述动态判断，不限于以上类别。
